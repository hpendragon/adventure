## Installation

Install WSL or Ubuntu and clone the project. In the root of the project run: python3 app.py

### Prerequisites

*   The game was created in Python3, therefore requires python3 to run.
*	Flask==2.0.1
*	python-dotenv==0.19.0

### Steps

1.  Clone the repository:
    ```bash
    git clone https://github.com/hpendragon/adventure
    ```
2.  Navigate to the project directory:
    ```bash
    cd your-repo-name
    ```
3.  Install necessary dependencies:
    ```bash
    npm install
    # or pip install -r requirements.txt
    ```

## Usage

Once you run app.py, open a browser and go to: http://127.0.0.1:5000/
